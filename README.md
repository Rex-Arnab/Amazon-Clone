# Installation Steps
# 1. Install the latest version of Node.js
# 2. Install the latest version of npm
## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
